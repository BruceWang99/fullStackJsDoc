性能优化

性能优化​

transform和transform3d​

  • 开启硬件加速

content-visibility: auto​

  • 告诉浏览器暂时跳过该元素的布局和渲染工作，直到这个元素滚动到当前视口
  • 兼容性较差

will-change​

  • 可以在行为触发之前告诉浏览器我们要进行一些什么样的变化操作，让浏览器好有个准备，独立图层，启动 GPU 为你渲染动画。
  • 兼容性还行
